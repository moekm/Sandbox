## To Compile With GCC:
Compiling the files is straightforward: install `GCC` and follow the instructions below.

> Tested and Working with gcc version 8.1.0 ``(x86_64-win32-sjlj-rev0, Built by MinGW-W64 project)``

```
$> gcc <FileName.c>
```

## To Compile With VSCode:

1. C/C++
2. Code Runner
